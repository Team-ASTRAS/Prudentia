INSTRUCTIONS

REAL TIME CONTROL

This mode allows the user to direct the Yaw, Pitch, and Roll of Prudentia in a remote control setting. Click on or hold down a button to drive Prudentia about the specified axis.

ATTITUDE INPUT

This mode allows the user to specify a target Yaw, Pitch, and Roll for Prudentia to orient to. Enter a Yaw, Pitch, and Roll in the boxes and click GO to start Prudentia.

SEARCH MODE

This mode allows the user to chose a location for the IR target that Prudentia will look for. Place the IR target 1.5m from the system’s center and click SEARCH to start Prudentia.

USE CAUTION DURING OPERATION! REMAIN 6ft AWAY FROM THE SYSTEM DURING OPERATION TO AVOID INJURY. PRESS ON THE EMERGENCY STOP BUTTON TO TURN OFF THE SYSTEM AT ANY TIME. READ THE FULL INSTRUCTION MANUAL BEFORE OPERATING THE SYSTEM.
