OPERATING MODES

🚀 ATTITUDE INPUT
This mode allows the user to specify a target Yaw, Pitch, and Roll for Prudentia to orient to. Enter a Yaw, Pitch, and Roll in the boxes and click GO to start Prudentia.

🚀 SEARCH
This mode allows the user to chose a location for the IR target that Prudentia will look for. Place the IR target 1.5m from the system’s center and click SEARCH to start Prudentia.

INTERFACE FEATURES AND INSTRUCTIONS

1. Read through the user manual before operating Prudentia.
2. Press the STOP button at the bottom of each page at any time to turn off Prudentia's motors.
3. To calibrate Prudentia, click the CALIBRATE button below and following the instructional prompts.
4. Once calibrated, you can access the Attitude Input and Search Mode pages using the navigation bar.
5. Click the STABILIZE button at the top right of each page at any time to reset Prudentia to 0°, 0°, 0° for yaw, pitch, and roll.
6. Prudentia has the option to record and save YPR, velocity, and YPR error data. Click the START LOGGING button to start recording data. Click the STOP LOGGING button to stop recording data. Click the CLEAR LOG button to erase recorded data. Click the DOWNLOAD button to save the data on your device.
7. Click the SHUTDOWN button at the top left of each page at any time to turn off the system.

USE CAUTION DURING OPERATION! REMAIN 6ft AWAY FROM THE SYSTEM DURING OPERATION TO AVOID INJURY. PRESS THE EMERGENCY STOP BUTTON TO TURN OFF THE SYSTEM AT ANY TIME. READ THE FULL INSTRUCTION MANUAL BEFORE OPERATING THE SYSTEM.
